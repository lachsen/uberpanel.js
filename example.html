• You can add arbitrary HTML into each UberPanel. This includes and and so on.

  • This UberPanel can be resized in both horizontally and vertically. Give it a try!

    • You can drag the bar to the right or below to change width or height respectively
    • You can also drag the lower right corner to change both width and height and the same time
  • This UberPanel has the 'fix-height' property. The user can only change the width. The height of the UberPanel will adapt automatically.
  • You can minimize Uberpanels by clicking the title bar. Clicking the title bar again will display the full Uberpanels again.
  • You can reorder Uberpanels by pressing the title bar and dragging the panel around.
    You can even move the panel from the left to the right!
  • The layout (position and size) of all Uberpanels can be saved into localstorage.
    When enabling autosave, the layout is saved whenever the user leaves the page. The layout can be saved manually by calling Uberpanels.saveState();

  • Visit the GitHub Project Page. Feedback, bug reports and suggestions are always welcome!

    Have fun with Uberpanels!
